# dcc-registry
DCC Member Issuer registry

> [!IMPORTANT]
>The DCC is only accepting registry entries for DCC members. All others wanting to register a DID that will verify in LCW or VerifierPlus should use the [Credential Engine registry](https://credentialengine.org/issuer-identity-registry/). Our [sandbox-registry](https://github.com/digitalcredentials/sandbox-registry) is also available for short-term testing.
